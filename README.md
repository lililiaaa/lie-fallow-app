# lie-fallow-app

项目简介：自己实现了一个浏览音乐，电影，图片功能的移动端

项目技术：使用 html,css,h5,c3 实现前端页面，使用 vue 框架进行开发实现页面交互，使用webpack 包

运行方法：

1.$ npm install    

2.$ npm run serve
